# Prompt Optimizer

## Description
This meta-prompt asks the AI to analyze and improve one of your existing prompts. It's a powerful way to learn prompt engineering and create a library of highly effective prompts for your specific use cases.

## Quick Version
```
Improve this prompt: [PASTE YOUR PROMPT]
```

## Max Power Version
```
You are a prompt engineering expert. Analyze the following prompt and rewrite it to be more effective, clearer, and more likely to produce high-quality results.

**Original Prompt:**
[PASTE YOUR PROMPT]

Provide:
1. **Improved Prompt:** The rewritten, optimized version.
2. **Key Changes:** A bulleted list of the specific changes you made.
3. **Reasoning:** Explain why each change improves the prompt.
4. **Additional Suggestions:** Any other tips for getting better results with this type of prompt.

Focus on clarity, specificity, and providing the right context and constraints.
```

## Example Usage

**To improve a vague prompt:**

"You are a prompt engineering expert... 

**Original Prompt:**
'Make a website for my business.'"

## Tips
- Use this to learn from your mistakes and improve over time.
- Create a collection of optimized prompts for recurring tasks.
- The AI can often spot ambiguities or missing context you didn't notice.
