# Images Folder

Place your images here according to the structure below.

## Quick Reference

- **Logo:** `logo/pow-logo.png` (or .svg, .jpg)
- **Bus Images:** `buses/[bus-name]-main.jpg`
- **Hero Background:** `hero/hero-background.jpg`
- **Event Images:** `events/[event-name].jpg`
- **Owner Photo:** `about/owner-photo.jpg`

## Folder Structure

```
images/
├── logo/          ← Your logo file goes here
├── buses/         ← Bus photos go here
├── hero/          ← Hero section background
├── events/        ← Event section images
├── about/         ← Owner/team photos
└── misc/          ← Any other images
```

**See `ASSETS_GUIDE.md` in the project root for detailed instructions.**
